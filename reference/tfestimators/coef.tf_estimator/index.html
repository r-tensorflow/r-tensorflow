Extract Model Coefficients

Extract Model Coefficients

# S3 method for tf_estimator
coef(object, ...)

Arguments

object

An estimator.

...

Optional arguments; currently unused.

Value

A named list of variables, such as each hidden layer's biases and weights matrices (if using any of the DNN estimators), global step, etc.

See also

Other custom estimator methods: estimator_spec, estimator, evaluate.tf_estimator, export_savedmodel.tf_estimator, predict.tf_estimator, train.tf_estimator
