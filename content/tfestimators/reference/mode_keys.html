The names for different possible modes for an estimator. The following standard keys are defined:

mode_keys()

Details

TRAINTraining mode.
EVALEvaluation mode.
PREDICTPrediction / inference mode.
TRAIN

See also

Other estimator keys: metric_keys, prediction_keys

Examples

# NOT RUN {
modes <- mode_keys()
modes$TRAIN
# }
