gs_local_dir(url, local_dir = "gs", echo = FALSE)

Arguments

url

Google Storage bucket URL (e.g. gs://<your-bucket>).
local_dir

Local directory to synchonize Google Storage bucket(s) to.
echo

Echo command output to console.
