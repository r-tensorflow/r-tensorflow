Use this step to create image embeddings columns from image columns.

step_image_embedding_column(spec, ..., module_spec)

Arguments

spec

A feature specification created with feature_spec().
...

Comma separated list of variable names to apply the step. selectors can also be used.
module_spec

A string handle or a _ModuleSpec identifying the module.
