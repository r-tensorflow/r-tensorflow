List of selectors that can be used to specify variables inside steps.

cur_info_env

Format

An object of class environment of length 0.

Selectors

  • has_type()

  • all_numeric()

  • all_nominal()

  • starts_with()

  • ends_with()

  • one_of()

  • matches()

  • contains()

  • everything()
