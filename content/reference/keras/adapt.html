Fits the state of the preprocessing layer to the data being passed.

adapt(object, data, reset_state = NULL)

Arguments

object

Preprocessing layer object
data

The data to train on. It can be passed either as a tf.data Dataset, or as an R array.
reset_state

Optional argument specifying whether to clear the state of the layer at the start of the call to adapt, or whether to start from the existing state. Subclasses may choose to throw if reset_state is set to FALSE. NULL mean layer's default.
