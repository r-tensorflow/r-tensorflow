The canonical set of keys that can be used to access metrics from canned estimators.

metric_keys()

See also

Other estimator keys: mode_keys, prediction_keys

Examples

if (FALSE) {
metrics <- metric_keys()

# Get the available keys
metrics

metrics$ACCURACY
