Installs the Google Cloud SDK which enables CloudML operations.

gcloud_install(update = TRUE)

Arguments

update

Attempt to update an existing installation.

See also

Other Google Cloud SDK functions: gcloud_init, gcloud_terminal

Examples

if (FALSE) {
library(cloudml)
gcloud_install()
}
